心理科學基礎統計

第4單元 這個分數是我要的嗎？單一樣本平均數的信賴區間

回顧第3單元的示範案例，有位學生收集12位同學的壓力感受分數。如果這位學生想知道他收集資料得到的平均數，是不是等於這間學校的所有學生的壓力感受。要完成這個目標的統計實務就是估計(Estimation)。

第3單元計算t檢定得到的平均數與效果量，都是一種點估計(point estimation)。信賴區間是點估計加上抽樣誤差範圍的上下邊界，邊界是平均值加上及減去臨界統計數與估計標準誤的乘積，臨界統計數是我們在分析前宣告的p值，所對應的統計數。如圖4.1顯示的兩個t分數。

圖 4.1: 雙尾檢定的p值在虛無假設抽樣分配的累積機率

APA格式規範已有明確的信賴區間報告格式，只是多數學期期刊與會議沒有要求必須報告。然而我們最好先認識效果量的信賴區間，因為如果有同一種研究的多次結果，效果量是比較所有結果的最佳統計數。

4.1 信賴區間的計算與報告

繼續第3單元的示範案例，我們先看如何使用統計軟體計算信賴區間。請再次開啟或下載單一樣本平均數t檢定的JASP示範檔與jamovi示範檔，只要重新開啟操作選單，像圖4.2一樣勾選Confidence Interval，就可以看到圖4.3的結果。你也可以看到同時能更改旁邊的百分比數值，這個數值是信賴水準，代表這次研究結果的信賴區間，有多少百分比的機率包括真正的母群平均數。從假設檢定被心理科學廣泛使用後，一直有學者主張信賴區間是實現統計思考更佳方法(Reichardt and Gollob 1997,Cumming and Finch (2005))。

圖 4.2: 單一平均數信賴區間操作設定示範。圖左是JASP，圖右是jamovi。

圖 4.3: 單一平均數信賴區間報表示範。圖左是JASP，圖右是jamovi。

你會注意到只有JASP可以計算效果量的信賴區間，其實jamovi也可以，只是放在另一個整合分析模組MAJOR。效果量的信賴區間在整合分析才會派上用場，在此讓你知道如何計算，是希望你了解所有統計量數的計算，或者統計量數的組合，都是一種估計。以下是示範案例加入信賴區間資料，依APA格式規範報告的結果：

  1. 12位大學生的壓力感受平均分數(M = 874.1666667, SD = 535.5618686; 95% CI [533.887 1214.446])顯著與原始報告不同，t(11) = -2.412， p = 0.035， d = -0.696。

  2. 12位大學生的壓力感受平均分數(M = 874.1666667, SD = 535.5618686; 95% CI [533.887 1214.446])顯著小於原始報告，t(11) = -2.412， p = 0.017， d = -0.696。

圖4.3是雙尾檢定的報表，如果你接著在單尾檢定操作選單勾選計算信賴區間，會發現下邊界是\(-\infty\)，上邊界(1151.817)也低於雙尾檢定的上邊界(1214.446)。這是因為使用單側臨界值計算所致，也顯示信賴區間與假設檢定的在目的上的不同，我們計算信賴區間是希望知道再做一次同樣的研究，能在多大的範圍以事先期望的機會(95%)，得到接近的結果。假設檢定是以最低容忍的風險，確保目前的結果是符合假設的。所以一般統計書示範信賴區間的計算公式，都是採用雙尾檢定臨界值。

所以我們最後要好好理解兩個問題，才能結束這個單元。第一個是事先設定的95%信賴水準是什麼意思？第二個是這一次研究得到的信賴區間，能保證真實的母群平均數有95%的機率在這個區間嗎？

4.2 信賴區間的機率本質

在第2單元裡我們已經知道，每次隨機程序獲得的平均數及變異數，會隨著樣本數目增加，趨近於母群的平均數與變異數，但是每一次隨機的結果都會有些許差別。信賴區間也是如此，即使樣本數目相同，每一次隨機抽樣或實驗的信賴區間，都不會一樣。來分析seeing thoery的區間估計動畫。

跑模擬動畫之前，可以在左邊頁面選擇抽樣分配模型、調整樣本數量(n)與信賴水準(\(1-\alpha\))。配合示範案例，建議選擇t分佈，調整樣本數量12，信賴水準0.95。按下Start Sampling，右方頁面會持續跑出代表獨立實驗的平均數與信賴區間的線條，你可以觀察一陣子，注意一下每一次的信賴平均數與信賴區間是不是都不一樣。如果有信賴區間包括預估的平均數\(\mu\)，線條是綠色。沒有的話，線條是紅色。

跑了一段時間，你可以按下Stop Sampling停止動畫。左方頁面有剛剛這段模擬結果的累積次數，你會發現有包含預估平均數的結果佔94%到96%，這就是信賴水準的真正意義。也因此有人主張信賴區間比假設檢定，更適合呈現統計思考的有效程度。

4.3 信賴區間與研究結果可再現性

那麼第一次研究結果的信賴區間，能不能包括之後條件一致的再現研究的平均數？Cumming與Maillardet提出捕獲百分比(capture percentag)的概念。他們提出一種模擬程序，實作之後發現各種條件的研究結果，第一次95%信賴區間只能捕獲之後約84%的研究結果平均數(Cumming and Maillardet 2006)。所以信賴水準與捕獲百分比是不一樣的事，但是現在許多有經驗的研究者還不夠了解。

Daniel Laken在個人部落格提供一個R程式碼，讓會用R的研究者能親自動手模擬，觀察捕獲百分比與信賴區間的不同。我了解本書讀者尚未正式接觸R，所以我先將Daniel Laken的程式碼改寫成以下內容，請你拷貝整段程式碼，貼到圖4.4的線上執行視窗，接著按一下視窗下方的Run it，等待幾秒鐘，看看模擬結果訊息是什麼？

n=12 #根據研究背景設定樣本數
nSims<-100000 #設定虛擬再現實驗次數

x<-rnorm(n = n, mean = 874.167, sd = 535.562) # 製造第一次實驗虛擬資料

#95%CI
CIU<-mean(x)+qt(0.975, df = n-1)*sd(x)*sqrt(1/n) # 計算第一次實驗信賴間上邊界
CIL<-mean(x)-qt(0.975, df = n-1)*sd(x)*sqrt(1/n) # 計算第一次實驗信賴間下邊界 

#製造虛擬再現實驗資料
CIU_sim<-numeric(nSims) 
CIL_sim<-numeric(nSims)
mean_sim<-numeric(nSims)

for(i in 1:nSims){ 
  x<-rnorm(n = 12, mean = 874.167, sd = 535.562) 
  CIU_sim[i]<-mean(x)+qt(0.975, df = n-1)*sd(x)*sqrt(1/n)
  CIL_sim[i]<-mean(x)-qt(0.975, df = n-1)*sd(x)*sqrt(1/n)
  mean_sim[i]<-mean(x) 
}

CIU_sim<-CIU_sim[CIU_sim<874.167]
CIL_sim<-CIL_sim[CIL_sim>874.167]

cat ( "根據第一次實驗資訊，再現的",nSims,"次新研究結果，其中有",(100*(1-(length(CIU_sim)/nSims+length(CIL_sim)/nSims))),"%的信賴區間包括樣本估計的平均數。")

mean_sim<-mean_sim[mean_sim>CIL&mean_sim<CIU]
cat("但是有",100*length(mean_sim)/nSims,"%的新研究平均數，落在第一次實驗的信賴區間",CIL,"與",CIU,"之間。")

圖 4.4: 線上R程式編譯器。請將程式碼複到視窗裡，再按Run it執行。

你可以多執行幾次，看看模擬結果報告有多少百分比的模擬結果信賴區間，包含預設平均數？第一次之後的模擬結果平均數，有多少百分比落入第一次的信賴區間？

前者你應該會發現幾乎都是95%左右，就像seeing theory的模擬動畫。後者你可能發現連續好幾次是84%左右，或者很多次超過90%，加上很多次低於60%。因此捕獲百分比是個浮動的數值，不過會遵守統計學知名的均值迴歸(regression to mean)，累積多次模擬會趨近84%。未來你接觸R，了解程式碼編寫規範後，可以再來試試更改各種條件，重新測試看看。

更重要的問題：如果有一批條件一致的研究結果要你分析，你認為信賴區間是好的指標嗎？我不會在這本書說明我的想法，因為這是你進階之後，才要面對的課題。
